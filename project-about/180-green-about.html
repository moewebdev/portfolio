Project Goal

Web development can get boring sometimes. So you gotta make it fun. This project basically is an inside joke between my girlfriend, myself, and my boss. I currently work for a vacation rental company that specializes in managing luxury properties. I decided it would be fun to create a parody listing that featured a property that no one in their right mind would ever want to rent. And thus 180 Green was born. It's still a work in progess. The site is 100% static and was not created using responsive design techniques. My goal was to have fun so I'll keep working on it here and there until I get bored and move on to another project.
