Project Goal

My goal for this project was to create a full-stack webpage that could be used by a local business to handle booking appointments and simple business inquires. To make things user friendly, I also created an admin application that the client can use to create, read, update, and delete data from the shop database.

Tech Stack used for this project

  • CSS3
  • Bootstrap4
  • jQuery
  • Ajax
  • MySQL
